# projeto_integrador_3

## Neste Projeto

Faremos um Jogo, utilizando o espaço da PUC Campinas para explorar e interagir com o app por meio das extensões Geolocator e google_maps_flutter

## Para começar

Utilize estes comandos no powershell para clonar na sua máquina:

- git clone LINK
- cd projeto_integrador_3
- flutter pub get
- flutter run
